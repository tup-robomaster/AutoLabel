# TUP目标检测数据集自动标注工具
## 环境配置
编译安装imagededup

git clone https://github.com/idealo/imagededup.git
## 使用说明
在该目录下打开终端,执行如下命令

1. 
   ```shell
   pip install -r requirements.txt
   ```
2. 
   ```shell
   cd Galaxy_Linux_Python_2.0.2106.9041/api/
   python setup.py install
   ```
3. 执行main.py,具体使用说明请参考--h参数
   ```shell
   cd ../..
   python main.py
   ```
